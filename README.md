ember-voca
==============================================================================

Ember Template helpers based on [voca.js](https://vocajs.com/)


Installation
------------------------------------------------------------------------------

```
ember install ember-voca
```


Usage
------------------------------------------------------------------------------

```hbs
{{voca-capitalize "hello world"}}
\\ => Hello World
```

Contributing

Compatibility
------------------------------------------------------------------------------

* Ember.js v3.12 or above
* Ember CLI v2.13 or above
* Node.js v10 or above
